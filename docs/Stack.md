## Stack.md

| Camada     | Tecnologia                                     | Função                        |
| ---------- | ---------------------------------------------- | ----------------------------- |
| Frontend   | HTML + CSS (AdminLTE) + Metabase Embed         | Interface e visualização      |
| Backend    | Python 3.11, Supabase Client, Pandas, Requests | Coleta e tratamento de dados  |
| Banco      | Supabase (PostgreSQL)                          | Armazenamento e automação     |
| ETL        | Make / Zapier / Python                         | Atualização diária de dados   |
| IA         | OpenAI API (GPT-4) + cache 24h                 | Insights automáticos          |
| Relatórios | ReportLab (PDF) + SMTP                         | Geração e envio de relatórios |
